# eMove Analysis Project

## Project Overview
This repository is dedicated to analyzing movement signatures in motion capture (MoCap) data that correlate with the emotional content of videos displayed through a VR headset. The project is a collaborative effort between Yoe and Simo.

## Repository Structure
- **RAWDATA/**: Directory for raw data files (excluded from version control).
- Under the RAWDATA there are subfolders for different types of DATA:
    - **MOCAP/**: Contains motion capture data files.
    - **EDA/**: Contains electrodermal activity data files.
    - **HR/**: Contains heart rate data files.
    - **STIMVIDEOS/**: Contains the stimulus videos shown during data collection.
    - **UNITYLOGS/**: Contains e.g. eye tracking data from the Unity application used during data collection.
- **README.md**: Project documentation.

## Notes
- Ensure that all raw data files are placed in the `RAWDATA/` directory.
- The `RAWDATA/` directory is excluded from version control to avoid issues with large files.
